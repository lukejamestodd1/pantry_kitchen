New Century Learning Centre

Company website
Company website for a freelance client

A friend and I built this website for a local Chinese language school. It is a dual language site which provides information for visitors, and has an administration area for staff to log in and publish news, photos, and updates. We built the back end using Node.js, Express, MongoDB, and Mongoose. We used Google Maps API to show campus locations dynamically when clicked, and Passport to manage user accounts. For the front end designs we used Materialize.css and converted these into jade layouts. We are still adding final touches so this project is not deployed yet. The repo is here

Gruel Intentions

Gruel Intentions

This site allows users to upload photos of their failed recipes. Users create an account or login with Facebook, make posts and can like other recipes. The backend is built with Rails and Sinatra, and I used Backbone, Mustache, and Bootstrap for the front. The site is hosted on Amazon Web Services and Heroku.

Hungr

Find friends who want the same thing to eat as you

Hungr

This web app lets you find nearby people who want to eat the same thing you do. Users login, choose a desired food or cuisine, and see a list of other users who want the same thing. The backend is built with Ruby on Rails, and I used Backbone.js and Mustache templating for the frontend single page app. Version 1 is deployed on Heroku but is still in development mode. So far, the Facebook login, change status, and view other users functions are working. I’m currently adding messaging functionality, user profiles, and creating a menu/control panel. After that I'll experiment with Facebook friend list data and use the Google Maps API to get user locations.

Trainer Buddy

A booking and schedule system for personal trainers

Trainer Buddy

This web app helps personal trainers organise their classes and bookings. The trainer adds a class to their schedule, and other users can then book a place. The app shows how many people are booked in for each class, and has a calendar of appointments for each user. Trainers keep a contact list of clients in the app, so they can send group messages to them easily. Customers can view trainer profiles and follow their favourites to stay updated.

This was a group project, and my role was to create the single page frontend app by integrating the html markup with the backend database tables. I created models, views, and the router in Backbone.js and used mustache templating to render the views into HTML.
