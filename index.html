New Century Learning Centre

Company website

Company website

A friend and I built this website for a local Chinese language school. It's a dual language site providing information for visitors, and has an administration area for staff to log in and publish news, photos, and updates.

We built the back end using Node.js, Express, MongoDB, and Mongoose.

We used Google Maps API to display campus locations dynamically, and Passport to manage user accounts.

For the front end designs we used Materialize and jade.

Gruel Intentions

A bad food network

Gruel Intentions

This site allows users to upload photos of their failed recipes. Users create an account and log in, and upload their chosen recipe with a picture. Users can "lick" and unlick their favourite recipes.

I built the backend with Ruby, Sinatra, and ActiveRecord. I used CarrierWave for image uploads and built the front using Bootstrap. The site is currently hosted on Amazon Web Services and Heroku.

I am currently making new pages so users can follow each other, and redoing the front designs with Materialize

Next I will add Facebook login functionality so users can sign in easily, and post recipes to Facebook

Hungr

Find friends who want the same thing to eat as you

Hungr

This web app lets you find people who want to eat the same thing you do. Users login, choose a desired food or cuisine, and see a list of other users who want the same thing who are nearby.

The backend is built with Ruby on Rails, and I used Backbone.js and Mustache templating to make the frontend single page app.

So far the Facebook login, get user locations, change statuses, and view other users functions are working.

I’m currently adding messaging functionality, user profiles, and creating a menu/control panel. After that I'll integrate Facebook friend list data to show which users are friends on Facebook.

Version 1 is deployed on Heroku but this project is very much a work in progress!

Trainer Buddy

A booking and schedule system for personal trainers

Trainer Buddy

This web app helps personal trainers organise their classes and bookings. The trainer adds a class to their schedule, and other users can then book a place. The app shows how many people are booked in for each class and has a calendar of appointments for each user.

Trainers keep a contact list of clients in the app so they can send group messages easily. Customers can view trainer profiles and follow their favourites to stay updated.

This was a group project. My role was to create the single page frontend app by integrating the html markup with the backend database tables. I created models, views, and the router in Backbone.js and used mustache templating to render the views into HTML.
